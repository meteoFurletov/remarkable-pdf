# Third-party assets

These files are vendored so the skill renders offline. Each keeps its own license.

- **marked** (`marked.min.js`) — Markdown parser. MIT License.
  https://github.com/markedjs/marked
- **KaTeX** (`katex/katex.min.css`, `katex/katex.min.js`, `katex/contrib/auto-render.min.js`,
  `katex/fonts/*`) — math typesetting. MIT License.
  https://github.com/KaTeX/KaTeX

`pypdf` (used for PDF bookmarks) is **not** bundled — install it with `pip install pypdf`
(BSD-3-Clause). https://github.com/py-pdf/pypdf
